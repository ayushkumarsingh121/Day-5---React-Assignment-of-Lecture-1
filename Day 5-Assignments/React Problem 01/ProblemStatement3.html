Reflection: As you complete the task, reflect on the differences between manipulating the DOM using traditional JavaScript and React's approach. Consider the benefits of React's declarative UI elements over direct DOM manipulation.

Reflection Points:

Traditional DOM Manipulation:

  • Directly updating HTML elements with methods like document.getElementById and innerHTML.
  • Can lead to complex, error-prone code when managing updates and state.

React's Approach:

  • Uses a declarative syntax where you describe what the UI should look like.
  • Automatically updates the UI when the state changes, reducing the need for manual DOM manipulation.
  • Makes the code more readable and easier to manage.
